# Weather App

## Описание

Приложение разработано в рамках изучения Compose. Основной функционал включает:
- Главный экран с погодой на текущее время
- Список погоды на целый день
- Список погоды на неделю

---

## Технологии

- `Kotlin`
- `Volley`, `AsyncImage`
- `Coroutines`, `Flow` / `LiveData`
- `MVI`
- `Compose`
